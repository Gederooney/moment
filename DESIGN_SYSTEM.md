# PodCut Design System\n\n## Vue d'ensemble\n\nLe design system PodCut est conçu pour créer une expérience utilisateur moderne, accessible et sûre pour la capture de moments dans les vidéos YouTube. L'interface est particulièrement optimisée pour une utilisation en voiture, avec des éléments de grande taille et un feedback visuel/haptique clair.\n\n## Principes de Design\n\n### 1. **Sécurité avant tout**\n- Boutons extra-larges (minimum 80px) pour une interaction sûre en conduisant\n- Contraste élevé pour une lisibilité optimale\n- Feedback haptique pour confirmer les actions sans regarder l'écran\n- Mode sombre automatique pour la conduite de nuit\n\n### 2. **Familiarité YouTube**\n- Utilisation du rouge YouTube (#FF0000) comme couleur primaire\n- Interface inspirée de l'écosystème YouTube pour réduire la courbe d'apprentissage\n- Patterns d'interaction cohérents avec les applications de média\n\n### 3. **Accessibilité universelle**\n- Conformité WCAG AA minimum\n- Labels d'accessibilité sur tous les éléments interactifs\n- Tailles de touch target minimum 44x44 points\n- Support des lecteurs d'écran\n\n### 4. **Performance et fluidité**\n- Animations de 300ms maximum\n- Micro-interactions subtiles pour le feedback\n- Transitions fluides entre les états\n- Optimisation pour React Native\n\n---\n\n## Palette de Couleurs\n\n### Couleurs
